# 月亮与六便士_毛姆_90

  1. 那些慷慨激昂的人自以为他们说的话是前人闻所未闻的，殊不知此类豪言壮语早已被说过上百遍，而且连说话的腔调也是大同小异。
  2. 那些作家当面称兄道弟，但转身就会用尖酸刻薄的言论相互挖苦，我常听得瞠目结舌。
  3. 他们就像是身体里的细胞，重要是很重要，但只要是健康的细胞，就会被巨大的总体吞没而显露不出来。
  4. 绝大部分的人做着普普通通的事情就心满意足了。
  5. 尽管心中有许多美好而奥秘的意思要表达，却只能借助会话手册上那些陈腔滥调。他们的大脑充满了想法，却只能告诉你园丁的姨妈的雨伞在屋子里。

# 伤心者_何夕_87

# 摘要

  1. 一种仇恨使人蔑视生命，另一种仇恨则使人顽强地活下去。
  2. 真正存在的只有一样东西，那便是能量以及能量的流动。
  3. 时光只有在回想的时候才发觉它过得真快。
  4. “你错了，改变世界的不是你们。” 何夕的声音变得有点异样。“是我。”
  5. 对于我们脚下这片浸透着苦难的古老土地来说。只有那些最最‘基本’的东西才会真正有用。
  6. 从某种意义上讲，地球上的所有生命都不过是一段各不相同的四进制程序代码罢了。

# 读后感

我觉得何夕和刘慈欣是国内最优秀的两位科幻作家，不同于刘慈欣的硬核，何夕的作品更多的是对于社会中软文化的思考，但是也同样充满想象的科幻色彩。何夕的文字更为流畅，读起来更舒服，但是感觉何夕有点太喜欢写爱情了，基本每篇都会出现男女主的感情故事，偶尔来一下还好，一直这样感觉有点水字数。

# 情人_[法] 玛格丽特。杜拉斯_83

  1. 我自以为我在写作，但事实上我从不曾写过，我以为我在爱，但我从来也不曾爱过，我什么都没有做，不过是站在那紧闭的门前等待罢了。
  2. 那笑声是再也听不到了，笑声，哭声，都听不到了。完了，完了，都忘了，都记不起来了。
  3. 我们憎恶生活，也憎恶我自己。
  4. 我再也不能得到了，我自以为还能，但是办不到了。

# 杀死一只知更鸟_哈珀。李_83

  1. 富林克林。罗斯福：“除了恐惧本身，他们没有什么可恐惧的。”
  2. 世界上就是有那么一类人，他们 —— 他们只顾担心来世，根本不去学习在今生如何做人。
  3. 我在接受他人之前，首先要接受自己。有一种东西不能遵循从众原则，那就是人的良心。
  4. 一个人很少能赢，但也总会有赢的时候。
  5. 一个人没必要把自己懂的东西都展现出来。
  6. 我不赞成以人的生命为代价保持虚伪的礼节。
  7. 如果人们能把事情归结于一个理由，就好办多了。
  8. 我之所以这样生活，是因为这就是我想要的生活。
  9. 他们以前做过，今天晚上又做了，将来还会在做，而且，他们这样做的时候… 似乎只有孩子会哭泣。
  10. 在我们生活的这个世界，总有什么东西让人丧失理智 —— 即使他们努力想做到公平，结果还是事与愿违。
  11. 我们的法庭也有缺陷，任何社会机构都不例外，但是，在这个国家里，我们的法庭是伟大的平等主义者。在我们的法庭里，人人生而平等。

# 茶花女_小仲马_85

  1. 如果不了解一个人的痛苦，而又想去安慰她，那实在是一件难事。
  2. 如果人的想象能够使感官带上某种诗意，而肉体的欲望有能在心灵的幻想前退却三分，那可真是一件值得庆幸的事！
  3. 爱情这件事，实在是一件极其孩子气的崇高行为。
  4. 男人们总是这样，都心急火燎地想知道那些听了以后会让他们心里痛苦的事。
  5. 人生只不过是为了反复满足一种持续不断的欲望，灵魂也只不过是维持爱情圣火的贞女。
  6. 人经常有一种孩子般的快乐，而破坏这种快乐的人是可恶的。
  7. 如果一个人不随着时间的推移而自我完整，那他也只是长了年龄而不长见识。
  8. 真正的爱情总是把人塑造得更完美，不管激起这种爱情的女人是怎么样的人。

# 呼兰河传_萧红_84

  1. 在乡村，人和动物一起忙着生，忙着死…
  2. 坟场是死的城郭，没有花香，没有鸟鸣，即使有虫，那都是唱奏着别离歌，陪伴着说不尽的死者永远的寂寞。
  3. 她无助的嘶狂着，圆眼睛望一望锁住的门不能自开，她不能逃走，事情必然要发生。
  4. 漫天星光，满屋月亮，人生何如，为什么这么悲凉。
  5. 人生为了什么，才有这样凄凉的夜。
  6. 他们想击退寒凉，因此带来了悲哀。

# 狂热分子_[美] 埃里克。霍弗_87

  1. 一个人愈是没有值得自夸之处，就愈容易夸耀自己的国家，宗族，种族或他所参与的神圣事业。
  2. 失业者宁愿追随贩卖希望的人，而不愿追随施与救济的人。
  3. 为最起码生活操劳的人，不会有时间，心情去悲愤或造梦。
  4. 一个人除非善于用脑子，否则自由就会成为他一种讨厌的负担。
  5. 当一个极权社会的人民揭竿而起时，他们要反对的不是它的邪恶，而是他的软弱。
  6. 爱国主义是歹徒最后的归宿。

# 棋王_阿城_89

书本的开头就十分引人入胜，通过 “我” 的观察，寥寥数笔就写出了棋王王一生最突出的两个特征，一是嗜棋如命，躲避着亲人的送别，却到处寻人下棋，一找到人下棋就开心的不得了；二是对吃的渴望，细节在于对其吃饭时的神态，最妙的是写王一生将桌子上的干了的饭粒抠出来再放入口中细细咀嚼的神态。

“若饭粒儿落在衣服上，就马上一按，拈进嘴里。若一个没按住，饭粒儿由衣服上掉下地，他也立刻双脚不再移动，转了上身找，吃完以后，他把两只筷子吮净，拿水把饭盒冲满，先将上面一层油花吸净，然后就带着安全到达彼岸的神色小口小口的呷，他对吃是虔诚的，而且很精细。有时你会可怜那些饭被他吃得一个渣儿都不剩，真有点儿惨无人道。”

用着最平淡的语气来写最传神的文章，这是这篇小说的精妙之处。而小说的高潮是王一生与九位棋赛胜者同时进行比赛。“他把嘴定在碗边儿，半响，回报了棋步，才咽一口水下去，“咕” 的一声，声音大的可怕，眼里有了泪花。” 没有写棋局的凶险，而去写喝水的情形来刻画说明。而王一生隐隐有股书呆子气，“脚卵” 送了自己珍藏多年的棋盘给棋赛领导来让王一生去参赛，他拒绝了，宁愿在棋赛结束后，去和棋赛的赢家再比一次。而且这篇小说充满着一种传统文化，例如 “势”，“阴阳” 等。风格有点类似于冯骥才的《俗世奇人》，也是用精妙的语言来刻画人物。小说可读性很强，情节引人入胜。


# 人类简史_[以色列] 尤瓦尔。赫拉利_87

  1. 农业革命并没有让大多数人都过上更好的生活，只是让人类以更低的标准养活更多的人。
  2. 学会用火，使人可以集中能量以供大脑的发展。
  3. 相信并不存在的事物，是人类最重要的技能。
  4. 历史的进程并不明智，有时甚至是错误接着一个错误。
  5. 历史的进程并不明朗，谁都无法准确的预测未来。
  6. 帝国，宗教客观上推动了全球化的进行，推动了艺术和科学的发展。
  7. 阶级使得大家有了共同语言和共同目标。
  8. 智人从未与自然和谐相处，从开始便是杀人凶手。
  9. 文字使得文化可以代代传承。
  10. 我们现在处于历史上最为和平的时代。
  11. 能源并不会被人类耗尽，只是需要工具。
  12. 环境问题是急需解决的问题。
  13. 资本主义冲掉所有抗拒全球化的力量。

# 撒哈拉的故事_三毛_88

  1. 我看见每个一个女人都用一片小石头沾着水，在刮自己的身体，没刮一下，身上就出现一条黑黑的浆汁的污垢，她们不太用水，要刮得全身的脏都松了，才用水冲。
  2. 光看这张照片，就令人眼花缭乱，招架不及，如果真人来了，加上香粉味一定更是精彩。
  3. 这么一个拘谨孤单年轻的心，碰到一点点即使是假的爱情，也当然要不顾一切地去抓住了。
  4. “我在想，飞蛾扑火时，一定是极快乐幸福的。”
  5. 人，是最怕认识自己的动物。
  6. 我认真地在想，关在牢里面的人，不一定比放在外面的人坏。
  7. 我并不气馁，人，多几种生活的经验总是可贵的事。
  8. 人，真是奇怪，没有外人来证明自己，就往往看不出自己的价值。
  9. 没有拍到红鹤自是可惜，但是那一刹那的美丽，在我的心底，一生也不会淡忘掉了。
  10. 没有变化的生活，就像是织布机上的经纬，一匹一匹的岁月都织出来了，而花色却是一个样子的单调。
  11. 路，像一条发光的小河，笔直地流在苍穹下。
  12. 写，是重要，而有时搁笔不写，却是更重要。
